01能源、航运与全球通胀截至 2026-07-23 08:30 北京时间

霍尔木兹、红海与黑海同时受阻，能源冲击从战事升级为运输和通胀问题

概览：市场面对的已不只是一次中东军事升级，而是三个关键能源通道同时降低可用性；油价、成品油、船期和保险成本开始共同上升。

Reuters与AP均报道，美伊冲突再度升级、霍尔木兹海峡仍大体关闭，胡塞力量同时威胁曼德海峡航运。WSJ进一步核算，波斯湾、红海和黑海受扰动的通道合计关系到约四分之一的全球石油供应；黑海方向的袭击还影响俄罗斯与哈萨克斯坦原油外运。

这条传导链已经出现实体价格：Brent在7月22日升至94美元上方，CMA CGM宣布自8月1日起对每个集装箱加收65—165美元紧急燃油附加费。世界银行首席经济学家称，若冲突持续六个月以上，最坏情景下2026年全球增长可能降至1.3%、全球通胀升至4.5%。这是情景测算而非基线预测，但说明能源冲击已进入增长、利率与债务压力的同一张表。

状态：航线受扰、油价和附加费均已发生；冲突持续时间与通道恢复速度仍未确定。
影响：原油 / 柴油与汽油 / 海运保险 / 全球通胀 / 美债收益率 / 高负债经济体。

02AI、芯片与资本开支2026-07-22 美股盘后

AI竞赛进入资产负债表阶段：Google与Tesla现金流承压，AMD用投资换取Anthropic采购

概览：AI需求仍在快速增长，但最新数字把问题从“谁的模型更强”改写为“谁能持续融资、供电并把巨额资本开支转化为现金流”。

Alphabet把2026年资本开支预期再次上调至1950亿—2050亿美元，Google Cloud收入同比增长82%至248亿美元，但FT报道其季度自由现金流为负59亿美元，为上市以来首次。Tesla收入增长26%至282.4亿美元，却因AI、机器人和产能投入录得11亿美元自由现金流流出，并把年度资本开支推至250亿美元以上。

同一天，AMD同意向Anthropic投资至多50亿美元，Anthropic则承诺采购数百亿美元AMD AI服务器、取得最高2吉瓦算力容量。交易既挑战Nvidia的供应地位，也暴露AI产业越来越明显的循环融资结构：芯片商给客户资本，客户再把资本转化为芯片与数据中心订单。

状态：Alphabet与Tesla为已公布财报；AMD—Anthropic为已宣布、按里程碑执行的投资与采购安排。
影响：云计算利润率 / 自由现金流 / AI芯片份额 / 数据中心电力 / 债务与股权融资 / AI估值。
